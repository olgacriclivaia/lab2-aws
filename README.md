---

### Ответ на вопрос:
**User Data** — это скрипт, который автоматически выполняется при запуске инстанса.

В данном случае он:
- обновляет систему
- устанавливает nginx
- запускает веб-сервер

**nginx** — это веб-сервер, который обрабатывает HTTP-запросы и отображает веб-страницы.
